Cicero Word Generator by akeshet

Intuitive exprimental control software for atomic physics experiments

Key Features: Control of National Instruments cards providing analog and digital outputs, using an intuitive graphical user interface. Control of other devices via RS232 ("serial") and GPIB commands. 1 - 2 microsecond time resolution on 60s long sequences, through use of variable frequency sample clocks to reduce buffer sizes. Client / Server architecture, separation of UI program from hardware output program, allowing scaling to arbitrary numbers of output channels. Batch processing, variable scanning, and run logging allow for automated data collection while eliminating tedious work.
